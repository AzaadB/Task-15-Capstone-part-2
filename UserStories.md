# User Stories for Pete's Car Emporium!

1. [ ] Replace current sticky note system
2. [x] Add a public facing page with basic contact info
3. [ ] Add an employee login to the cars app
4. [x] Provide a welcome page after login
5. [ ] Provide easy navigation
6. [ ] Display current user and assigned role
7. [x] Provide a logout option
8. [ ] Require users to login at least once per week
9. [x] Cars are assigned to specific employees
10. [x] Cars have a ticket #, title, note body, created & updated dates
11. [x] Users can be Employees, Managers, or Admins
12. [ ] Cars can only be deleted by Managers or Admins
13. [ ] Anyone can create a note (when customer checks-in)
14. [ ] Employees can only view and edit their assigned cars
15. [ ] Managers and Admins can view, edit, and delete all cars
16. [ ] Only Managers and Admins can access User Settings
17. [ ] Only Managers and Admins can create new users
